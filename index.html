GPT Summarizer

A natural language processing app that summarizes long PDF documents using OpenAI’s GPT API. Users upload a PDF, and the app parses, chunks, and generates human-like summaries in seconds.

  • Built with Python, OpenAI API (v1), PyMuPDF, and Streamlit
  • Handled token limitations by implementing dynamic text chunking and prompt formatting
  • Reduced manual document reading time by 90% — ideal for students, researchers, and professionals
GitHub Live Demo

ScopeAI

ScopeAI is an intelligent insight generator designed to extract meaningful knowledge from various sources—>PDFs, YouTube videos, and news articles, with ease. Powered by OpenAI's GPT-3.5, it provides automatic summarization, sentiment analysis, named entity recognition, and generates insightful follow-up questions. The clean, interactive Streamlit UI, enriched with custom CSS, ensures an intuitive and responsive user experience.

  • Tech stack: Python, GPT-3.5 (OpenAI), VADER, PyMuPDF, yt-dlp, Whisper, BeautifulSoup, Streamlit, Custom CSS
  • Features: Multi-source parsing, NLP analysis, responsive UI
  • Deployed via Streamlit Cloud with wide compatibility and smooth performance
